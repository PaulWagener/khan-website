Setting up accounts

In order to get the most out of Khan Academy, you'll need to set up student and teacher accounts. We've created a handy guide to make sure you'll be up and running in no time.

We recommend setting up Google accounts to register classes on Khan Academy. You can register accounts through Google Apps for Education.

  • Google Apps for Education registration
  • Google Apps for Education common questions
A smooth start to class

To ensure that class is able to begin on the right track, we recommend pre-registering students for class with assigned Google accounts (e.g., john-appleseed@school.org for the student John Appleseed) and passwords (e.g., school2012).
