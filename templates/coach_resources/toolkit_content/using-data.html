Using data

How can data translate into classroom practices? Click on the report to see how the data can be used in your classroom.

Just in case: If you’re not already familiar with each KA report and feature, check out this nifty video.

Progress Summary

  • During class: use the filtering ability to hone in on a specific exercise. Using the student data, assign peer tutors, hold a small group seminar, or directly tutor an individual.
  • The night or week before: check to see if there are specific exercises that groups of students might need help with and prepare to teach small group seminars in those exercises; set class goals based on this data (ex. By the end of this week, I want everyone to move from the unstarted column to the proficient column)
  • Weeks before the next unit: use the filter to look up exercises in the upcoming unit. Use the student data to determine which students need to focus on which concepts.
